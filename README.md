## Web Explorer

The repository includes a local web UI to explore coded guideline outputs.

### Features

- Full-text search across all coded segments and guideline metadata.
- Filtering by `B` and `B_label` across all available files in `outputs/`.
- Cross-guideline code overview (code frequency and coverage).
- Single-guideline reader with color-highlighted coded segments.

### Run locally

From the repository root:

```powershell
uv run scripts/build_web_data.py
```

Then start a local static server:

```powershell
python -m http.server 8000
```

Then open:

- `http://localhost:8000/web/`

The app requires `web/data/guidelines-web.json` generated by the build command above.

### Rebuild after new outputs

Whenever new coded files are added in `outputs/`, rebuild the web dataset:

```powershell
uv run scripts/build_web_data.py
```
